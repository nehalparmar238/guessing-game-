Multi-Language Guessing Game

A simple number guessing game implemented in C, C++, and Python.  
This project is perfect for comparing syntax and logic across different programming languages while practicing Git and GitHub basics.

---------------------------------------------------------------------------------------------------------------------------------

Features
-> Random number generation and user input handling  
-> Feedback for guesses (too high / too low)  
-> Implemented in three different languages:
    c (guessing_game.c)
    C++ (guessing_game.cpp)
    Python (guessing_game.py)
